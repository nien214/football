How to Play

  • Click a footballer to select it.
  • Mouse input: move mouse for direction, click to select/pass/shoot.
  • Keyboard input: move using keyboard cursor keys or W,A,S,D.
  • V.Stick input (mobile portrait): use stick to move, shoot button to pass/shoot.
  • Press keyboard 1, 2, or 3 to select footballer 1, 2, or 3.
  • Press Spacebar to pass/shoot toward the mouse or keyboard cursor.
  • Press C to mute/unmute crowd sound.
  • If your footballer has the ball, press 1/2/3 to pass to footballer 1/2/3.
  • Opponent can steal if the ball is near them, except 1 second protection after possession.
  • Win by scoring more goals before full time.
